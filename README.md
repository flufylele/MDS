# MDS
Proiect MDS

Facultativ: User stories , geneza listei de cerinte (cum s-a renuntat la anumite lucruri)
Obligatoriu: Lista de cerinte + Descriere interfata / interfete
Schema interna , diagrama clase , diagrama functii la fiecare clasa 
Comentarii in cod (pe functii, pe variabile, pe clase) recunoscute de platforma ( pun /// )
Protocoale de test (fiecare pe cate o functie si fiecare sa contina mai multe teste diferite)
Protocoale de testare (copie la protocolul de test + inserat dupa fiecare task rezultatul testului, cu albastru pozitiv, cu rosu negativ)
Fisa de mentenanta (recomandare + masura)
